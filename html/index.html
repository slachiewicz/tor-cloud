Tor Cloud

Tor Cloud on Amazon EC2

The Tor Cloud is a user friendly way of deploying relays to help users in countries securely access the Internet, free of censorship. By donating your bandwidth to the Tor Network, you improve the safety and speed at which users can access the Internet. Learn more about Tor »

This project runs on the Amazon EC2 cloud computing platform, which powers Amazon.com and other major websites such as Netflix. Users are able to launch their own virtual machines and computing resources with flexible and cost-efective terms. The Tor network can always benefit from more volunteer bandwidth. Learn more about Amazon EC2 »

AWS Free Usage Tier

750 hours of Amazon EC2 Linux Micro Instance usage, enough hours to run continuously each month.

* visit http://aws.amazon.com/free/ for details.

Don't qualify for the Free Tier? No problem, our images are configured to limit usage at $30/month based on Amazon fees.

How to Get Started

1. Sign-up for an AWS account and subscribe to EC2 service: Sign Up Now!
(Amazon will send you an email once your account is activated.)

2. Click and start your Tor Cloud bridge on one of the many locations available:

  • us-east-1
    Virginia
  • us-west-1
    N. California
  • ap-northeast-1
    Tokyo
  • eu-west-1
    Ireland
  • ap-southeast-1
    Singapore
